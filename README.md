# Awesome Pyrange, Awesome Code
# 一个让你起飞的代码自动生成工具
一键生成 Java Controller, Service, Model, Mapper, Mock 以及 Vue 前端代码

# 工具编译环境要求
jdk: 11  
idea: 2021.3.3

# 打包方法
Gradle -> Tasks -> intellij -> buildPlugin

# 调试方法
Gradle -> Tasks -> intellij -> runIdeForUiTests  
使用 debug 模式运行

# 插件试用 Idea 版本
2020.1.2 -> 2022.2.1
